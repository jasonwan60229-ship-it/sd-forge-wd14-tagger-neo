WD14 Tagger Neo for SD-WebUI (Forge Compatible)此擴充功能為 Stable Diffusion WebUI Forge 版本提供了強大的 WD14 (Waifu Diffusion 1.4) Tagger 功能。它允許您上傳圖片，使用多種 ViT/ConvNext/SwinV2/EVA02 模型進行標籤分析，並將生成的標籤一鍵傳送到文生圖或圖生圖的提示詞框中。✨ 特性多模型支持： 內建支持 wd-vit-tagger-v3, wd-convnext-tagger-v3 等多種 Tagger 模型。ONNX 高速推論： 使用 ONNX Runtime 進行推論，實現高效能、低延遲的標籤生成。Forge/WebUI 兼容： 兼容最新的 SD WebUI Forge 和標準 WebUI 架構。一鍵傳送： 快速將生成的標籤傳送到 Txt2Img 或 Img2Img 分頁。GPU 優先： 預設嘗試使用 CUDA (GPU) 進行加速推論。🛠️ 安裝與模型下載1. 擴充功能安裝在 WebUI 中，進入「擴充功能 (Extensions)」->「從網址安裝 (Install from URL)」。輸入此 GitHub 儲存庫的網址。點擊「安裝」。回到「已安裝 (Installed)」分頁，點擊「應用並重啟 UI (Apply and restart UI)」。2. 模型下載 (重要！)由於模型檔案較大，本擴充功能不包含模型。您必須手動下載 ONNX 模型和對應的 CSV 標籤文件，並將其放入：extensions/sd-forge-wd14-tagger-neo/models/所需檔案列表：| 模型名稱 | ONNX 檔名 | CSV 檔名 || :--- | :--- | :--- || wd-vit-tagger-v3 | wd-vit-tagger-v3.onnx | wd-vit-tagger-v3.csv || wd-convnext-tagger-v3 | wd-convnext-tagger-v3.onnx | wd-convnext-tagger-v3.csv || wd-swinv2-tagger-v3 | wd-swinv2-tagger-v3.onnx | wd-swinv2-tagger-v3.csv || wd-eva02-large-tagger-v3 | wd-eva02-large-tagger-v3.onnx | wd-eva02-large-tagger-v3.csv |您可以在 Hugging Face 找到這些檔案。💻 使用方法點擊 WebUI 頂部的 WD14 Tagger Neo 分頁。上傳您想要分析的圖片。在下拉選單中選擇您想使用的 Tagger 模型。調整「標籤閥值 (Threshold)」，閥值越高，輸出的標籤越少、越精確。點擊「開始分析 (Interrogate)」。標籤將顯示在右側的文字框中。您可以使用下方的按鈕一鍵傳送標籤到其他分頁。⚠️ 疑難排解錯誤：onnxruntime 未安裝：請確保您的網路連接正常，install.py 應該會在啟動時自動安裝。錯誤：模型文件缺失：請檢查您是否將所有 .onnx 和 .csv 文件正確放在 models/ 資料夾中，並且檔名完全匹配。
