---------------------------------------------------------------

WD14 Tagger for sd-webui-forge-neo

This WD14 Tagger extension was developed specifically for sd-webui-forge-neo.

IMPORTANT: Manual Model Download Required

Please remember to download the following ONNX model files from SmilingWolf's Hugging Face page:

Model File

Download Link

wd-convnext-tagger-v3.onnx

https://huggingface.co/SmilingWolf/wd-convnext-tagger-v3

wd-eva02-large-tagger-v3.onnx

https://huggingface.co/SmilingWolf/wd-eva02-large-tagger-v3

wd-swinv2-tagger-v3.onnx

https://huggingface.co/SmilingWolf/wd-swinv2-tagger-v3

wd-vit-tagger-v3.onnx

https://huggingface.co/SmilingWolf/wd-vit-tagger-v3

Placement:

You must place all these files into the following folder:

\extensions\sd-forge-wd14-tagger-neo\models

Note: The filenames MUST match the names listed above for the extension to work correctly.

P.S. I haven't figured out how to write the automatic model download function yet. If anyone in the community knows how to implement this, please feel free to contribute!
